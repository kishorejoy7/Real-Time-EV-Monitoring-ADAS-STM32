# 🚗 Real-Time EV Monitoring & ADAS Dashboard using STM32

A real-time Embedded Systems project that monitors Electric Vehicle parameters and implements basic ADAS safety features using **STM32 Blue Pill**, **PICSimLab**, and a **Python Dashboard**.

## ✨ Features

- 🚗 Live Speed Monitoring
- 🔋 Battery (SOC) Monitoring
- 🌡️ Motor Temperature
- ⚡ Torque Display
- 🛑 Brake & Accelerator Monitoring
- 📡 Front & Blind Spot Detection
- ⚠️ Collision Warning
- 📈 Live Speed Graph
- 🖥️ Real-Time Python Dashboard

## 🛠️ Technologies

- STM32F103C8T6 (Blue Pill)
- Embedded C
- STM32CubeIDE
- Python
- PySerial
- Matplotlib
- PICSimLab

## 🚀 How to Run

1. Open the project in **STM32CubeIDE**.
2. Build and flash the firmware.
3. Run the Python dashboard:
   ```bash
   pip install pyserial matplotlib
   python dashboard.py
   ```
